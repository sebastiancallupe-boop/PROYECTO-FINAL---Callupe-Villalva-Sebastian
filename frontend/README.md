# RapidoCourier Frontend

Panel web estatico para consumir el API Gateway.

## Ejecutar

Desde la carpeta del proyecto:

```powershell
cd frontend
python -m http.server 5173
```

Luego abrir:

```text
http://localhost:5173
```

El gateway debe estar activo en:

```text
http://localhost:8080
```

Usuarios iniciales:

```text
admin / Admin123
operador / Operador123
cliente / Cliente123
```
